Sitio web para el aprendizaje de las Etiquetas básicas de HTML, atributos y valores.

El sitio tiene como propósito enseñar de forma más interactiva al usuario los diferentes temas relacionados con las etiquetas básicas de HTML,
atributos y valores. Se tienen en cuenta los diferentes aspectos, también se observarán imágenes relacionados con el tema.

Etiquetas, atributos y elementos de HTML

HTML se define de un total de 91 etiquetas, de las cuales 10 se consideran obsoletas.
Sin embargo, a veces una etiqueta se considera obsoleta por si sola ya que no contiene la suficiente información
para estar completamente definida, por lo que para eso contamos con los atributos: pares nombre-valor separados por un =
y escritos en la etiqueta inicial de un elemento después del nombre del elemento.
Como ejemplo tenemos la siguiente estructura general de una linea de codigo en el lenguaje HTML:

tag attribute1="value1" attribute2="value2">content

Tipos de atributos

Aunque cada una de las etiquetas HTML define sus propios atributos, encontramos algunos comunes a muchas o casi todas
las etiquetas, que se dividen en cuatro grupos según su funcionalidad:

Atributos básicos

Atributos de internacionalización

Atributos de eventos

Atributos de foco

Elementos

Además de etiquetas y atributos, HTML define el termino elemento para referirse a partes que componen los documentos HTML.
El lenguaje HTML clasifica a todos los elementos en dos grupos: Elementos en línea o inline elements y elementos en bloque.
La diferencia entre ambos viene dada por el modelo de contenido por el formato y la dirección. La principal diferencia entre ambos
es la forma en la que ocupan el espacio disponible en la pagina. Apreciaremos toda esta información más detalladamente en el siguiente video:
